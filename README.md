AI Job Application Command Centre

This project was developed using Bolt AI Builder for the Gappy AI "Ship to Get Hired" Hackathon.

Live Demo

<https://ai-job-application-c-ildz.bolt.host>Project Description

AI-powered platform that helps students and recent graduates:

- Track job applications.
- Analyze resumes against job descriptions.
- Generate recruiter outreach messages.
- Prepare for interviews.
- Monitor application progress from a centralized dashboard.

Tech Stack

- Bolt AI Builder
- React
- TypeScript
- Tailwind CSS
